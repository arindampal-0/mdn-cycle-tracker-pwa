# Cycle Tracker
MDN Tutorial PWA Cycle Tracker
https://developer.mozilla.org/en-US/docs/Web/Progressive_web_apps/Tutorials/CycleTracker

## Setup

```shell
npm install
npm start
```
or
```shell
pnpm install
pnpm start
```

### Dev Setup

```shell
npm install
npm run dev
```
or
```shell
pnpm install
pnpm dev
```
